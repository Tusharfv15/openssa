PROBLEM-SOLVING AGENT with Planning, Reasoning, Knowledge & Informational Resources¶

This module contains OpenSSA’s main Agent class, which brings together agentic Planning and Reasoning capabilities, domain-specific Knowledge and Informational Resources to solve complex problems.

An agent’s problem-solving process involves a Plan that can be created by either LM-automated generation or expert guidance, and the execution by the agent’s Reasoner of that Plan, either dynamic (i.e., with as-needed further decomposition) or static (i.e., literally per the Plan as-is).

Users should pick the combination of plan creation and plan execution that is most appropriate for the availability of expert knowledge and structured expert plans in the concerned domain.
