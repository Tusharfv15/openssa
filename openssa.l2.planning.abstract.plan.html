ABSTRACT TASK PLAN INTERFACE¶

AbstractPlan is OpenSSA’s abstract base class for problem-solving task plans.

A plan has a target task to solve, and can contain decomposed sub_plans for solving that task.

A plan can be executed through its own .execute(…) method, by a specified Reasoner that optionally takes into account some given domain-specific Knowledge and/or other results. The plan execution returns a string result.
