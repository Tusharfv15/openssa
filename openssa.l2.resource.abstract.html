ABSTRACT INFORMATIONAL RESOURCE INTERFACE¶

AbstractResource is OpenSSA’s abstract base class for informational resources that problem-solving agents access.

An informational resource is simply something that has a globally-unique name (within the running program), has a potentially non-unique but informationally helpful name, and can .answer(…) given questions with string responses.
